Позиционирование элементов

Положение элементов относительно системы координат называют позиционированием. Типы позиционирования: асболютное, относительное, фиксированное, нормальное или статическое. Z-индексы - позиционирование работает, когда задано одно из свойств позиционирования: absolute, relative or fixed.

Создала index_p.html файл (Пример 1). Три абзаца с текстом сгенерировала на сайте http://lipsum.com

Свойства прописала в style_p.css (Пример 2).

Красная картинка над текстом шириной 650px и высотой 75px.

Как и в коде (Пример 4) картинка встала перед абзацами с текстом (Пример 5), занимая целую строку

Абсолютное позиционирование

Теперь в html файле допишем для картинки идентификатор id="new", а в css пропишем для него правило, не забывая при этом про фигурные скобки 

                    #new {
                            position:absolute;
                            top: 0;
                            left: 0;
                    }

В css файле (Пример 7) пропишем значения top and left

Видим, что картинка (Пример 8) позиционируется относительно окна браузера, 50px по вертикали и 50px по горизонтали. В итоге получается, что если нет других дополнительных условий, то свойство position absolute, дает право элементу позиционироваться, относительно окна браузера.

И если стоит цель спозиционировать в верхний левый угол (Пример 10), то значениям top and left поставить 0, т.е. от верха и от левого края. Всё получилось!

Посмотрим, что еще можно делать с абсолютным позиционированием. В html файл добавим новый див контейнер (Пример 11).

В css файле (Пример 12) для идентификатора "container" зададим абсолютное позиционирование, ширину и высоту, фоновый цвет

В файле css для идентификатора "container" задали top: 0; right: 0; Светлая картинка-блок (Пример 14) улетает в верхний правый угол.

А теперь картинку поместим в див контейнер (Пример 15).

В итоге получаем, что если элемент с абсолютным позиционированием находится внутри другого элемента с абсолютным позиционированием, то он начинает позиционироваться не относительно окна браузера, а относительно своего родителя.
